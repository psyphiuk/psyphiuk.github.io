Layout Ideas

This section contains a number of themed layout templates based on display: flex; or display: grid;, all exploring the idea of using minimal css classes and keeping the css DRY. For most of this page each section, example and elements uses these template class to create separation ( gap, flex-gap or grid-gap).

The following templates show the results of a set of classes that all begin with grid- which automatically includes an enabler class that adds the display: grid; to the container. So for the even columns example below all that is needed is to add the class grid-even-columns to use the style.

Responsive even columns

Note: You control the number of columns simply by including child elements, no other styling is required. With this style if there is not enough space for the columns to fit in then any that don't fit will wrap to the next row.
